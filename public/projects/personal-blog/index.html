Building My Personal Blog with Hugo

This project documents the process of creating my personal blog using Hugo, a fast and flexible static site generator.

Tech Stack

  • Static Site Generator: Hugo
  • CSS Framework: Custom SCSS
  • Hosting: GitHub Pages
  • Version Control: Git

Key Features

  1. Dark/Light Mode Toggle
  2. Category-based Content Organization
  3. Responsive Design
  4. Math Equation Support (KaTeX)
  5. Code Syntax Highlighting
  6. Fast Loading Times
